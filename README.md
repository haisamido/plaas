# PLaaS
Plotting as a Service (PLaaS)

## How to build
docker build -t plaas haisamido/plaas

## How to run
docker run -d -p 2222:22 plaas

## How to plot
ssh -t root@localhost -p 2222 "/usr/bin/gnuplot -p -e 'set terminal dumb;plot sin(x)'"
